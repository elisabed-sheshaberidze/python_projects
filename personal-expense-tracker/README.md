# Personal Expense Tracker

## About This Project
The project assists users to organise their expenses by categorizing them
and displaying statistics.

## What Will It Do?
- Register users and store their data using SQLite
- Authenticate users to access their accounts
- Manually enter expenses and income (stored in an SQLite database)
- Categorize expenses and display summaries

## Why This Project?
It helps me practice advanced topics like OOP and databases, giving me
practical experience implementing theoretical concepts in real-world projects.
