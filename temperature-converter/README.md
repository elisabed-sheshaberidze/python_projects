# 🌡️ Temperature Converter

Temperature Converter accepts any temperature unit — Celsius, Kelvin, 
or Fahrenheit. The program validates the user's input and converts 
it to the other two temperature units.

## Example
Input: `100C`  
Output: `100C is equal to 212.00 Fahrenheit and 373.15 Kelvin.`

## Concepts Used
`str` `float` · loops · functions · input validation

## How to Run
python temperature_convertor.py
